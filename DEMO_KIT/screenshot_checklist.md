# Screenshot Checklist

Use this checklist when refreshing screenshots for outreach, README updates, or portfolio materials.

## Setup

- Enable demo mode.
- Reset demo data.
- Confirm the Dashboard shows the full 15-inquiry sample story.
- Use a clean browser window.
- Avoid showing private files, terminal output, or real customer information.

## Required Screenshots

- Dashboard with populated KPIs.
- Patient Inquiries tab with diverse statuses.
- Weekly Summary with Practice Performance Snapshot.
- Export tab with download buttons and export preview.

## Naming

Use these names when replacing screenshot assets:

```text
dashboard.png
inquiries.png
weekly_summary.png
export.png
```

## Quality Check

- Text is readable.
- No Streamlit error banners are visible.
- Estimated Treatment Value is visible on the Dashboard.
- Follow-up urgency is visible.
- Sample data looks realistic and fake.
- Browser chrome is cropped only if it improves clarity.
