## Objection Responses

### We already have an EHR.

That makes sense. This is not meant to replace an EHR. It is a lightweight front-office view for tracking inquiries and follow-ups before someone becomes an active patient.

### We use spreadsheets.

That is exactly where this can help. It keeps the simplicity of a spreadsheet but adds a cleaner Practice Performance Dashboard, overdue follow-up review, weekly summaries, and patient-friendly exports.

### We do not need another tool.

Understood. The goal is not to add complexity. The demo is useful only if it helps the practice owner quickly see follow-up gaps, active patients, and estimated treatment value.

### Is this HIPAA compliant?

The current version is a demo MVP using fake sample data. It is not positioned as a production HIPAA system yet. A live version would need appropriate hosting, access controls, policies, and compliance review.

### How much does it cost?

The starter demo setup begins at $500. A practice-ready launch package starts around $1,500 depending on setup, hosting, and workflow polish needs.
