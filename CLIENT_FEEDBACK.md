# Client Feedback Log

Use this file after outreach calls and demos. Keep feedback specific, dated, and tied to buyer language.

## Feedback Template

```text
Date:
Practice / Contact:
Role:
Demo stage:

What resonated:
What confused them:
Workflow pain mentioned:
Objections:
Pricing reaction:
Requested changes:
Follow-up action:
Decision / next step:
```

## Starter Feedback Themes To Watch

- Missed or delayed follow-ups
- Lack of visibility into inquiry sources
- Difficulty knowing which inquiries became patients
- Concern about replacing existing tools
- Concern about HIPAA or patient data handling
- Desire for a simple front-desk workflow
- Need for owner-level weekly reporting

## Demo Questions To Ask

- Does this match how your office receives patient inquiries?
- Where would this fit into your current front-desk workflow?
- Which KPI would you look at first every morning?
- What would stop you from using this?
- What would make this worth paying for?
