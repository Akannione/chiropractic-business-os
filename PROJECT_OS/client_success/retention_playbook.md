# Retention Playbook

Use this playbook to keep chiropractic clients engaged after setup.

## Monthly Check-In Process

1. Review client health status.
2. Check whether weekly reviews are happening.
3. Ask whether follow-ups are being completed.
4. Identify one friction point.
5. Capture feedback and objections.
6. Confirm next action and owner.

Monthly questions:

- Are you still reviewing follow-ups each week?
- Which KPI do you look at first?
- What feels useful?
- What feels confusing?
- What is still happening outside the system?

## Quarterly Review Process

1. Prepare KPI review.
2. Review inquiry trends.
3. Review conversion trends.
4. Review follow-up performance.
5. Identify wins and challenges.
6. Set next-quarter goals.
7. Confirm whether the client is a testimonial, referral, or case study candidate.

Use `quarterly_business_review_template.md` for the call.

## Renewal Discussion Process

1. Start with outcomes, not features.
2. Review usage consistency.
3. Review follow-up improvements.
4. Discuss current pain points.
5. Confirm whether the system still fits.
6. Present renewal or support options clearly.

Renewal positioning:

- Keep inquiry follow-up visible.
- Keep the owner informed weekly.
- Prevent the workflow from drifting back to scattered notes or spreadsheets.

## Referral Request Process

1. Ask only if the client is Green or strongly Yellow and improving.
2. Ask for one specific introduction.
3. Explain the type of practice that fits.
4. Keep the ask low-pressure.
5. Track referral source in `PROJECT_OS/outreach/outreach_tracker.csv`.

## Retention Warning Signs

- Weekly review stops.
- Front desk is unclear who owns updates.
- Owner stops checking the dashboard.
- Follow-ups become overdue repeatedly.
- Client asks for unrelated features instead of using the current workflow.

## Recovery Process

1. Ask what stopped usage.
2. Remove unnecessary complexity.
3. Restart with one habit: review follow-ups weekly.
4. Confirm whether the system still solves a real problem.
5. Decide whether to continue, adjust scope, or end cleanly.
