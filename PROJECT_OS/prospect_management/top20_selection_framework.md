# Top 20 Selection Framework

Use this framework to consistently identify the highest-priority 20 chiropractic practices from a pool of 100.

## 1. Scoring Workflow

1. Add each practice to the prospect tracker.
2. Complete baseline research.
3. Score the practice with `qualification_scorecard.md`.
4. Assign a tier using `targeting_framework.md`.
5. Add notes about red flags and strong fit signals.

Required score:

- 80-100: Tier 1
- 65-79: Tier 2
- 50-64: Tier 3
- Below 50: Low Priority

## 2. Qualification Workflow

Evaluate each prospect across:

- Practice size
- Monthly inquiry volume
- Current tracking method
- Follow-up process
- Reporting maturity
- Openness to technology

When public information is missing, mark the item as unknown instead of guessing aggressively.

## 3. Re-Scoring After Discovery Calls

After a call, update the score based on what the practice confirms:

- inquiry volume
- current tracking method
- follow-up ownership
- weekly reporting process
- decision-maker clarity
- technology openness
- actual pain level
- timing and budget fit

Move the prospect up or down a tier if the new score changes.

## 4. Ranking Methodology

Rank the 100 prospects in this order:

1. Qualification score.
2. Tier.
3. Follow-up pain.
4. Inquiry volume.
5. Owner or office-manager accessibility.
6. Website and review activity.
7. Technology openness.
8. Lack of red flags.

If two practices have the same score, prioritize the one with clearer inquiry volume and easier decision-maker access.

## 5. Outreach Prioritization

Use this order:

1. Tier 1 practices with clear contact information.
2. Tier 1 practices with strong online activity but unclear decision-maker.
3. Tier 2 practices with strong follow-up pain signals.
4. Tier 2 practices with strong website and review signals.
5. Tier 3 practices only if they have a specific reason to engage.

The final top 20 should include:

- 10-15 Tier 1 practices.
- 5-10 strong Tier 2 practices.
- 0-3 Tier 3 practices only if they have unusually strong signals.

Do not include Low Priority practices in the first 20.
