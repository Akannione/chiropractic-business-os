# Discovery Call Scorecard

Use this immediately after a discovery call.

## Call Details

```text
Prospect name:
Practice name:
Contact role:
Call date:
```

## Scores

```text
Qualification score:
Pain score:
Urgency score:
Buying likelihood:
```

Scoring guidance:

- Qualification score: use `PROJECT_OS/prospect_management/qualification_scorecard.md`.
- Pain score: 1-5 based on missed follow-up, scattered tracking, and weak reporting.
- Urgency score: 1-5 based on how soon they want the problem solved.
- Buying likelihood: Low, Medium, High, or Unknown.
