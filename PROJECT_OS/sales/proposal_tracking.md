# Proposal Tracking

Use this template to track proposals after demos.

| Practice Name | Proposal Date | Package | Proposal Value | Status | Follow-Up Date | Outcome | Notes |
|---|---|---|---:|---|---|---|---|
| Example Chiropractic | 2026-06-15 | Practice Launch | 1500 | Sent | 2026-06-18 | Open | Waiting for owner review. |

## Status Options

- Draft
- Sent
- Viewed
- Questions Asked
- Negotiating
- Accepted
- Declined
- Paused

## Outcome Options

- Open
- Closed Won
- Closed Lost
- Follow-Up Later

## Notes Guidance

Track:

- Main objection.
- Response used.
- Decision-maker.
- Timing concern.
- Selected package.
- Next action.
