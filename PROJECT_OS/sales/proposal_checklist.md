# Proposal Checklist

Use this before sending a proposal to an interested chiropractic practice.

## Required Before Sending

- [ ] Discovery completed.
- [ ] Qualification score completed.
- [ ] Demo completed.
- [ ] Proposal package selected.
- [ ] Pricing confirmed.
- [ ] Onboarding packet ready.

## Proposal Quality Check

- [ ] Proposal uses current pricing only.
- [ ] Scope does not promise EHR, billing, insurance, scheduling, payments, automation, or patient portal features.
- [ ] Current practice challenges are specific to the prospect.
- [ ] Recommended solution is simple and practice-owner friendly.
- [ ] Timeline is clear.
- [ ] Next steps are easy to understand.

## Send Notes

```text
Practice:
Contact:
Package:
Proposal date:
Follow-up date:
Main objection to address:
```
