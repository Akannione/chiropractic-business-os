# Discovery Call Flow

Use this flow to keep calls simple and consultative.

```text
Introduction -> Discovery -> Qualification -> Pain Identification -> Demo Invitation -> Next Steps
```

## Introduction

Set the tone:

- Thank them for the time.
- Explain this is a fit conversation.
- Clarify that the tool does not replace EHR, scheduling, billing, or insurance systems.

## Discovery

Learn the current workflow:

- How inquiries arrive.
- Who manages them.
- How they are tracked.
- How follow-up happens.
- What the owner reviews weekly.

## Qualification

Map answers to:

- Practice Size
- Inquiry Volume
- Tracking Method
- Follow-Up Process
- Reporting Maturity
- Technology Openness

Use the qualification scorecard after the call.

## Pain Identification

Clarify what matters:

- Missed follow-ups.
- Scattered notes or spreadsheets.
- Owner uncertainty.
- Limited weekly reporting.
- Front-desk workflow friction.

## Demo Invitation

If there is fit, ask:

```text
Based on what you described, would it be useful to see a short walkthrough of how this dashboard tracks inquiries and follow-ups?
```

If there is not fit, say so clearly and avoid pushing.

## Next Steps

Choose one:

- Send demo website.
- Send ROI calculator.
- Schedule live demo.
- Send proposal after demo.
- Set follow-up date.
- Mark no fit.
