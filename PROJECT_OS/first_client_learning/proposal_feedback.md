## Proposal Review Questions

- Was the recommended package clearly tied to the discovery call?
- Did pricing create confusion?
- Did the scope boundaries feel clear?
- Did the prospect ask for EHR, scheduling, billing, insurance, or unrelated functionality?
- What should be clarified before the next proposal?
