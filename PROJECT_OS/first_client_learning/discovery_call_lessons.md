# Discovery Call Lessons

Use this file after each real discovery call.

## Call Entry

```text
Prospect:
Practice:
Date:
Qualification score before call:
Qualification score after call:
Decision maker present:
Inquiry volume:
Current tracking method:
Follow-up process:
Reporting process:
Objections:
Questions asked:
Confusing areas:
Lessons learned:
Recommended change:
Next step:
```

## Learning Questions

- Which questions produced the most useful information?
- Which questions felt too technical?
- Which objections appeared more than once?
- Did the qualification score improve after the call?
- Did the call naturally lead to a demo?
