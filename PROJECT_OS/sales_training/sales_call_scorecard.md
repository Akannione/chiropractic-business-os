# Sales Call Scorecard

Use this after practice calls to improve outreach quality. Score each section from 1 to 5.

## Call Details

```text
Date:
Practice:
Contact:
Role:
Call type:
Next step:
```

## Score Sections

| Section | Score 1-5 | Notes |
|---|---:|---|
| Clarity |  | Was the explanation simple and practice-owner friendly? |
| Confidence |  | Did the call sound calm, direct, and credible? |
| Pricing Explanation |  | Was pricing explained without apology or pressure? |
| EHR Boundary Explanation |  | Was it clear this does not replace the EHR? |
| Privacy Explanation |  | Were demo-data and production-data limits explained honestly? |
| ROI Explanation |  | Was value tied to fewer missed inquiries, faster follow-up, and weekly visibility? |
| Next-Step Close |  | Did the call end with a clear next action? |

## Red Flags

- The explanation sounded too technical.
- The buyer thought the app was an EHR or scheduler.
- Privacy or HIPAA questions were answered too casually.
- Pricing was discussed before the buyer understood the problem.
- No next step was agreed.
