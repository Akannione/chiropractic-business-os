# First Client Milestones

Use this document to track the first paying-client path from outreach through testimonial.

## Milestone 1: First Reply

Success criteria:

- A chiropractor or practice manager responds to outreach.
- Response is logged in the outreach tracker.
- Objection, interest level, or requested next step is captured.

Evidence to save:

- Date of reply.
- Prospect name.
- Message summary.
- Next action.

## Milestone 2: First Discovery Call

Success criteria:

- Discovery call is scheduled and completed.
- Current inquiry process, follow-up process, and reporting process are documented.
- Qualification score is updated after the call.

Evidence to save:

- Discovery notes.
- Updated qualification score.
- Objections.
- Next step.

## Milestone 3: First Demo

Success criteria:

- Demo is completed with realistic demo data.
- Prospect understands what the system does and does not do.
- Prospect identifies whether inquiry tracking and follow-up visibility are relevant.

Evidence to save:

- Demo date.
- Post-demo feedback form.
- Questions asked.
- Interest level.

## Milestone 4: First Proposal

Success criteria:

- Proposal is sent after discovery and demo.
- Package and pricing are clear.
- Scope boundaries are included.
- Follow-up date is scheduled.

Evidence to save:

- Proposal date.
- Package offered.
- Proposal value.
- Follow-up date.

## Milestone 5: First Signed Client

Success criteria:

- Client accepts one of the existing packages.
- Payment/process next step is agreed.
- Onboarding packet is sent.

Evidence to save:

- Package selected.
- Acceptance date.
- Main contact.
- Onboarding start date.

## Milestone 6: First Onboarding

Success criteria:

- Setup, data import, training, and go-live steps are completed.
- Client knows how to enter inquiries, update statuses, review follow-ups, view weekly summary, and export data.
- Handoff document is provided.

Evidence to save:

- Deployment notes.
- Training attendees.
- Go-live date.
- Open support questions.

## Milestone 7: First Testimonial

Success criteria:

- Client has used the system long enough to provide honest feedback.
- Testimonial request is made without pressure.
- Any public claim is accurate and non-guaranteed.

Evidence to save:

- Client quote.
- Permission status.
- Measurable outcome if available.
- Success story draft.
