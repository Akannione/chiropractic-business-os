# Outreach Metrics

Track these metrics weekly while building the chiropractic outreach pipeline.

## Outreach Volume

Number of prospects contacted during the period.

```text
Outreach Volume = count of prospects with initial outreach sent
```

## Response Rate

Percentage of contacted prospects who replied.

```text
Response Rate = replies / contacted prospects * 100
```

## Demo Booking Rate

Percentage of contacted prospects who booked a demo.

```text
Demo Booking Rate = demos booked / contacted prospects * 100
```

## Proposal Rate

Percentage of demos or qualified conversations that led to a proposal.

```text
Proposal Rate = proposals sent / demos or qualified conversations * 100
```

## Close Rate

Percentage of proposals that became closed-won deals.

```text
Close Rate = closed won / proposals sent * 100
```

## Review Guidance

- Track counts first; do not overcomplicate reporting.
- Focus on patterns in objections, demo requests, and follow-up timing.
- Use the metrics to improve outreach quality, not just volume.
