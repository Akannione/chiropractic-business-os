# Outreach Status Definitions

Use these statuses in `outreach_tracker.csv` to keep outreach consistent.

| Status | Meaning |
|---|---|
| Not Contacted | Prospect has been identified but no outreach has been sent. |
| Contacted | Initial outreach has been sent or a first call/message happened. |
| Follow-Up Sent | At least one follow-up message has been sent after initial contact. |
| Demo Scheduled | A demo or discovery call has been booked. |
| Proposal Sent | Pricing or paid pilot scope has been sent. |
| Negotiating | Prospect is discussing terms, timing, scope, or objections. |
| Closed Won | Prospect agreed to buy, pilot, or proceed with paid work. |
| Closed Lost | Prospect declined, went unresponsive after enough attempts, or is not a fit. |

## Usage Notes

- Use one current status per prospect.
- Update `last_contact_date` after every outreach touch.
- Set `next_follow_up_date` before ending each outreach session.
- Record objections and responses honestly.
