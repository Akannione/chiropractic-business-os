# Sales Assets

This folder contains optional outreach and demo materials for the Chiropractic Business OS. These assets are not required to run the Streamlit app and do not change the application schema or functionality.

## Contents

- `demo_video.mp4`: narrated 3-4 minute walkthrough video using the current demo screens.
- `demo_video_narration.md`: source narration script for the walkthrough video.
- `objection_handling.md`: honest responses to common chiropractor objections during outreach.
- `screenshots/dashboard.png`: dashboard screenshot with populated KPIs.
- `screenshots/inquiries.png`: patient inquiry workspace screenshot.
- `screenshots/weekly_summary.png`: weekly summary and snapshot screenshot.
- `screenshots/export.png`: export tab screenshot.
- `one_page_offer.pdf`: one-page offer for chiropractic practices.
- `faq.pdf`: buyer-facing FAQ.
- `pricing.pdf`: simple pricing sheet.
- `discovery_questions.pdf`: discovery call question list.
- `demo_script.pdf`: short walkthrough script.
- `onboarding_checklist.pdf`: practice onboarding checklist.

## Usage

Use these files during outreach, discovery calls, and client demos. The narrated video can be sent before a call or used as a backup walkthrough when a live demo is not practical. Pair it with the `demo_script.pdf` for live calls.

## Trust Notes

Use conservative language during outreach. The Chiropractic Business OS helps practices see inquiries in one place, improve follow-up consistency, reduce missed opportunities, and review estimated treatment value. It does not guarantee revenue or replace clinical, billing, insurance, or scheduling systems.
