# Handoff Checklist

Use this checklist after setup, demo, or deployment.

## Files And Access

- [ ] Share repository or deployment URL as appropriate.
- [ ] Share README run instructions.
- [ ] Share relevant sales or training PDFs.
- [ ] Share screenshot folder if needed.
- [ ] Confirm who owns the database path and hosting account.

## Documentation

- [ ] Review environment variables.
- [ ] Review demo mode behavior.
- [ ] Review sample data replacement instructions.
- [ ] Review SQLite persistence limitations.
- [ ] Review troubleshooting section.

## Client Workflow

- [ ] Confirm who adds inquiries.
- [ ] Confirm who updates follow-up dates.
- [ ] Confirm who reviews Weekly Summary.
- [ ] Confirm export use case.
- [ ] Confirm next feedback date.

## Boundaries

- [ ] Clarify no EHR integration is included.
- [ ] Clarify no scheduling is included.
- [ ] Clarify no payment processing is included.
- [ ] Clarify production compliance requirements before using real patient data.
