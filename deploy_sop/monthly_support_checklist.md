# Monthly Support Checklist

Use this checklist for light maintenance or retainer support.

## App Health

- [ ] Confirm app starts successfully.
- [ ] Confirm Dashboard loads.
- [ ] Confirm Weekly Summary loads.
- [ ] Confirm CSV exports work.
- [ ] Confirm database path is writable.

## Data Review

- [ ] Confirm inquiry count looks expected.
- [ ] Confirm overdue follow-ups are visible.
- [ ] Confirm active patients are tracked.
- [ ] Confirm estimated treatment value is reasonable.
- [ ] Confirm no malformed dates are breaking views.

## Client Feedback

- [ ] Ask what confused users this month.
- [ ] Ask which KPI was most useful.
- [ ] Ask whether exports were used.
- [ ] Capture feature requests in `FEATURE_REQUESTS.md`.
- [ ] Capture feedback in `CLIENT_FEEDBACK.md`.

## Sales And Growth

- [ ] Review whether current pricing still fits value delivered.
- [ ] Identify any client-specific workflow improvements.
- [ ] Avoid adding major features without paid justification.
- [ ] Update screenshots if the UI changed.
