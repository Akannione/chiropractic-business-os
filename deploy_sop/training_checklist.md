# Training Checklist

Use this checklist during a practice owner or office manager walkthrough.

## Opening

- [ ] Explain this is not an EHR.
- [ ] Explain this is not a scheduling or billing system.
- [ ] Explain the app tracks patient inquiries and follow-ups.
- [ ] Confirm demo data is fake.

## Dashboard Training

- [ ] Review Total Patient Inquiries.
- [ ] Review Follow-Ups Needed.
- [ ] Review Overdue Follow-Ups.
- [ ] Review Active Patients.
- [ ] Review Estimated Treatment Value.
- [ ] Open KPI Help and explain definitions.

## Patient Inquiry Workflow

- [ ] Add a fake patient inquiry.
- [ ] Review inquiry details.
- [ ] Update status.
- [ ] Set next follow-up date.
- [ ] Show dashboard recalculation.

## Weekly Summary

- [ ] Open Weekly Summary.
- [ ] Review Practice Performance Snapshot.
- [ ] Download snapshot text or CSV if useful.

## Export

- [ ] Download patient inquiry CSV.
- [ ] Download follow-up CSV.
- [ ] Explain exports use patient-friendly headers.
